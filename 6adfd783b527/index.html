给你的QQNT装上插件-LiteLoaderQQNT

发表于2025-01-14|更新于2025-01-17|软件
|浏览量:

这是什么

LiteLoaderQQNT 是 QQNT 的插件加载器，一般在 QQNT 的环境内简称为 LiteLoader
它可以让你自由地为 QQNT 添加各种插件，并实现例如美化主题、增加功能等各种功能
支持 QQNT 桌面端 全架构 最低 25765 版本到官网最新版本（一般情况是这样的）

安装

下载文件

你需要先把LiteLoaderQQNT下载到你的电脑上。

  • 打开Releases页面页面，下载LiteLoaderQQNT.zip文件，解压放到任意位置，也可以通过git克隆项目仓库，命令如下：

如果你用的是Windows设备，还需要额外的移除文件校验的步骤。

  • 前往DllHijack ，下载适合你设备的文件，放到QQ.exe同级目录下

修改文件手动安装

先找到 app 文件夹的路径，修改里面的两个文件

  • 查看 QQNT 根目录，是否存在 versions 文件夹

    • 是，则路径为 QQNT\versions\版本号\resources\app
    • 否，则路径为 QQNT\resources\app

  • 在 app 目录中，创建一个文件，修改一个文件

    • 创建 app/app_launcher/*.js 文件，其中 * 为文件名可随意设置，写入 require(String.raw*) 其中 * 为 LiteLoaderQQNT 的路径
    • 修改 app/package.json 文件，将 main 后面的路径改为 ./app_launcher/*.js 其中 * 为你刚才创建的文件名

检查安装

打开QQ设置，查看是否有LiteLoaderQQNT选项，rt。

安装插件列表

默认情况下安装插件需要去官网查找挨个下载，这样的方式非常不人性化，你可以通过安装LL-plugin-list-viewer这个插件来可视化的下载并管理你的插件。

  • 下载
    前往LL-plugin-list-viewer下载list-viewer.zip 文件，无需解压。
  • 加载插件
    打开你的 LiteLoaderQQNT设置页面,点安装插件并选择下载的文件，然后重启QQ，回到设置页面，就可以看到插件列表了。
